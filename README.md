# FacetWP - Post Status

## Description

This plugin add post status sources for your facets.

## Installation

### Requirements

* FacetWP 3.0.0 or greater

### Installation

1. Unpack the download-package.
2. Upload the files to the /wp-content/plugins/ directory.
3. Activate the plugin
